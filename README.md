# Mueeza Portfolio Website

A complete professional portfolio website built with **Flask**, **HTML5**, **CSS3**, and **JavaScript**. Features a modern dark theme, glassmorphism design, particle background, typing animation, fully responsive layout, and a working contact form with SQLite database.

## Features

- ✨ Modern dark theme with glassmorphism cards
- 🎯 Animated typing effect on homepage
- 🌟 Particle.js-style animated background
- 📱 Fully responsive (mobile, tablet, desktop)
- 🧭 Smooth scrolling & page transitions
- 📝 Working contact form with SQLite storage
- 🎨 Gradient highlights and hover effects
- 📂 Projects showcase with 9 portfolio items
- 🔗 Social media links integrated

## Tech Stack

- **Backend:** Python Flask
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Icons:** Font Awesome 6
- **Deployment Ready:** GitHub, Render, PythonAnywhere, VPS
